File xplhandlers/x10_basic.lua

xPLGirder is a Girder component to connect Girder to an xPL network.

 This file is an xPL message handler for x10.basic messages. Received message with this schema will be transformed into a standard Girder event. The event source will be xPLGirder. The event payloads will be;

  1. x10 address
  2. command
  3. xpl message type (cmnd, trig, stat)
  4. nil
